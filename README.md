# Flask To-Do App

## 📌 Description
This is a simple To-Do web application built using Flask.  
It allows users to add and delete tasks using a clean Bootstrap interface.

---

## 🚀 Features
- Add tasks
- Delete tasks
- Simple UI using Bootstrap
- Flask routing and templates

---

## 🛠️ Technologies Used
- Python
- Flask
- HTML
- Bootstrap
